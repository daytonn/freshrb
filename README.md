# Freshrb

A Ruby Framework for the paranoid ruby developer who values configuration over convention.

## Installation

Add this line to your application's Gemfile:

    gem 'freshrb'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install freshrb
